# 年齢層別比較機能の設計

## 🎯 なぜ年齢調整が重要か

### 統計的バイアスの問題
```
在留外国人の年齢構成:
20-29歳: 35% (労働・留学目的)
30-39歳: 28% (技能実習・就労)
40歳以上: 37%

日本人全体の年齢構成:  
20-29歳: 11%
30-39歳: 13% 
40歳以上: 76% (高齢化社会)
```

**⚠️ 問題**: 若い世代の方が犯罪率が高いのは世界共通の傾向
**📊 解決**: 同年齢層での比較が統計的に公正
